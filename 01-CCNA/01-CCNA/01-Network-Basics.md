1. End Devices
End Devices (also called End Hosts) are the devices that send and receive data on a network. Examples include PCs, laptops, servers, printers, and mobile phones. These are the devices that users directly interact with. Every network communication starts from one end device and ends at another end device.

2. Switch
A Switch is a Layer 2 networking device used to connect multiple devices within the same Local Area Network (LAN). It forwards data to the correct destination using MAC addresses. A switch does not require an Internet connection to work. It allows devices such as PCs, printers, and servers to communicate with each other inside the same office or building.

3. Wireless Access Point (WAP)
A Wireless Access Point (WAP) provides wireless (Wi-Fi) connectivity to devices such as laptops, smartphones, and tablets. The access point is connected to the switch using an Ethernet cable. It allows wireless devices to join the same LAN and communicate with wired devices.

4. Router
A Router is a Layer 3 networking device that connects different networks together. It uses IP addresses to forward data between networks. A router is required to access the Internet or communicate with another network. Without a router, devices can communicate only within the same LAN.

5. Firewall
A Firewall is a security device that protects a network from unauthorized access and cyber threats. It monitors incoming and outgoing network traffic and allows or blocks traffic based on security rules. Firewalls help protect company data from hackers, malware, and other attacks.

6. Local Area Network (LAN)
A Local Area Network (LAN) is a network that connects devices within a limited geographical area, such as an office, school, home, or university campus. LANs are usually fast, reliable, and inexpensive. Switches are the primary devices used to build a LAN.

7. Wide Area Network (WAN)
A Wide Area Network (WAN) connects multiple LANs that are located in different cities, states, or countries. For example, connecting a company's Chennai office with its Bangalore office creates a WAN. Routers are used to connect different LANs over a WAN.

8. Network Topology
Network Topology is the physical or logical layout of a network. It shows how devices such as switches, routers, servers, and computers are connected. Common topology types include Star, Bus, Ring, Mesh, and Tree. A topology diagram helps network engineers understand and manage the network.

9. Network Speed
Network Speed refers to how quickly data is transferred from one device to another. It is measured in Mbps (Megabits per second) or Gbps (Gigabits per second). Higher network speed results in faster file transfers, smoother video calls, and better application performance.

10. Network Cost
Network Cost is the total expense required to build and maintain a network. The cost includes networking devices such as switches, routers, firewalls, access points, cables, servers, software licenses, Internet services, and maintenance. Larger networks generally have higher costs.

11. Network Security
Network Security is the practice of protecting a network from unauthorized access, cyber attacks, malware, and data theft. Security is achieved using technologies such as firewalls, intrusion prevention systems (IPS), antivirus software, VPNs, multi-factor authentication (MFA), and access control policies.

12. Intrusion Prevention System (IPS)
An Intrusion Prevention System (IPS) is a security device that monitors network traffic to detect malicious activities and cyber attacks. Unlike a firewall, an IPS can identify suspicious behavior and automatically block attacks before they reach the network.

13. Availability
Availability means ensuring that the network and its services are always accessible when users need them. A highly available network minimizes downtime by using backup devices and redundant connections. High availability is critical for businesses that operate 24/7.

14. Scalability
Scalability is the ability of a network to grow as the organization expands. A scalable network allows new users, devices, servers, or branch offices to be added without redesigning the entire network. Good network planning makes future expansion simple and cost-effective.

15. Reliability
Reliability is the ability of a network to operate continuously without frequent failures or interruptions. A reliable network provides stable communication, minimizes downtime, and ensures that users can access network services consistently.

16. Single Point of Failure (SPOF)
A Single Point of Failure (SPOF) is any single device or component whose failure causes part or all of the network to stop working. For example, if an office has only one router and it fails, users will lose Internet access. To avoid SPOF, organizations deploy redundant devices such as backup routers, switches, and firewalls.

17. Redundancy
Redundancy is the practice of installing backup devices or connections to ensure continuous network operation. If the primary router, switch, firewall, or Internet link fails, the backup device automatically takes over. Redundancy improves both availability and reliability.
